# 20xx_td3

Este repositorio va a contener todo el material de la cátedra para el año vigente. Se espera que cada alumno de TD3 pueda contar con una copia de este repositorio para recibir actualizaciones de material, ejercicios, trabajos prácticos y evaluaciones.

Este repositorio funcionará como plataforma de entrega para **todas** las actividades que se desarrollen en la cátedra.

## Cómo copiar el repositorio

Con una cuenta personal de GitHub asociada a la organización _utn-fra-td3_, vamos a poder acceder a este repositorio y poder crear una copia o hacer un [fork](https://docs.github.com/es/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo).

Para hacer un fork, vamos al botón de _Fork_ o _Bifurcación_ en la esquina superior derecha del repositorio:

<div align="center">

![fork button](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRgLFbgZtkHYSfO4LMZI1IefPGTu3-X163HjA&s)

</div>

Luego de eso, vamos a ver un menú donde elegiremos el nombre del repositorio.

<div align="center">

![create fork](https://i0.wp.com/user-images.githubusercontent.com/90000203/162800388-9c07b590-75c4-4a86-8042-6eddea94bb61.png?ssl=1)

</div>

El nombre que le daremos en _Repository name_ seguirá esta nomenclatura: **20xx_td3_5cc_apellido_nombre** donde:

* _xx_ será el año de cursada.
* _cc_ será la comisión (11 o 12).
* _apellido_ será el primer apellido del alumno.
* _nombre_ será el primer nombre del alumno.

En cuanto al _Owner_, tenemos que asegurarnos que figure nuestreo nombre de usuario.

Una vez que le damos a _Create fork_ ya tenemos nuestro repositorio forkeado y listo para trabajar!
